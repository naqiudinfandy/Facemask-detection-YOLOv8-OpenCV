🛡️ Face Mask Detection

Detect face mask usage in real-time using AI-powered YOLOv8.

📌 About This Project

This project uses YOLOv8, OpenCV, and Flask to detect face mask usage in real-time. It categorizes individuals as:

  • ✅ Proper Mask: Mask is worn correctly.
  • ⚠️ Incorrect Mask: Mask is worn improperly.
  • ❌ No Mask: No mask detected.

🛠 Technologies Used

Backend:
Python (Flask, OpenCV, YOLOv8)

Frontend:
HTML, CSS, JavaScript

Machine Learning:
Deep Learning for object detection

📊 Evaluation Metrics

  • 🎯 Precision: 0.8981
  • 📈 Recall: 0.8483
  • 🏆 mAP50: 0.9053
  • 📊 mAP50-95: 0.6355
